cd (Change Directory):

    Change the current working directory.
    Example:
    cd Documents

ls (List):

    List files and directories in the current directory.
    Example:
    ls -l

pwd (Print Working Directory):

    Display the current working directory.
    Example:
    pwd

cp (Copy):

    Copy files or directories.
    Example:
    cp file.txt new_directory/

mv (Move):

    Move or rename files and directories.
    Example:
    mv file.txt new_location/

rm (Remove):

    Remove files or directories.
    Example:
    rm file.txt

mkdir (Make Directory):

    Create a new directory.
    Example:
    mkdir new_directory

rmdir (Remove Directory):

    Remove an empty directory.
    Example:
    rmdir old_directory

touch:

    Create an empty file or update the access and modification times of a file.
    Example:
    touch new_file.txt

nano (Text Editor):

    Open the nano text editor.
    Example:
    nano file.txt

cat (Concatenate):

    Display the contents of a file.
    Example:
    cat file.txt

chmod (Change Mode):

    Change the permissions of a file or directory.
    Example:
    chmod +x script.sh

ps (Process Status):

    Display information about running processes.
    Example:
    ps aux

kill (Terminate a Process):

    Terminate a process by specifying its process ID (PID).
    Example:
    kill -9 PID

top:

    Display a dynamic view of system processes.
    Example:
    top

git init:

    Initialize a new Git repository.
    Example:
    git init

git clone:

    Clone a repository into a new directory.
    Example:
    git clone https://github.com/example/repository.git

git add:

    Add changes in the working directory to the staging area.
    Example:
    git add file.txt

git commit:

    Record changes in the repository.
    Example:
    git commit -m "Commit message"

git status:

    Show the status of changes as untracked, modified, or staged.
    Example:
    git status

git pull:

    Fetch changes from a remote repository and merge them into the current branch.
    Example:
    git pull origin main

git push:

    Push local changes to a remote repository.
    Example:
    git push origin main

git branch:

    List, create, or delete branches.
    Example:
    git branch feature_branch

git checkout:

    Switch branches or restore working tree files.
    Example:
    git checkout feature_branch

git merge:

    Merge changes from one branch into another.
    Example:
    git merge feature_branch

git log:

    Display the commit history.
    Example:
    git log

git diff:

    Show changes between commits, commit and working tree, etc.
    Example:
    git diff

git reset:

    Reset the current HEAD to a specified state.
    Example:
    git reset --hard HEAD^

git remote:

    Manage remote repositories.
    Example:
    git remote -v

git fetch:

    Download objects and refs from another repository.
    Example:
    git fetch origin
